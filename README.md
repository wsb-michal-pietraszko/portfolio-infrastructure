Nie jestem pewien czy to trzeba bylo wykonac.

1. Aby uruchomic serwer nalezy wykonac:
    ```bash 
    $ ./setup.sh git@github.com:wsb-michal-pietraszko/portfolio-backend.git
    ```
2. Pipenv shell
   ```bash
   $ cd portfolio-backend/
   $ pipenv shell 
   $ cd ..
   ```
3. Start
    ```bash
   $ ./start.sh 
    ```
